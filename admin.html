Album Admin

How to add a new album
  1. Enter your Admin token below. Once per browser.
  2. Click "+ New album". Add the title, date, type and description, set the location, then Save.
  3. Add photos: open "Open folder in Drive", upload your photos, then Refresh and pick a cover.

Stored only in this browser. Required for any change.
